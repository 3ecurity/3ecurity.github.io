Windows Powershell下载文件的几种方式

Apr 11, 2019 • Powershell,Download

下载：

1. Net.WebClient

powershell.exe -Command "& {(New-Object Net.WebClient).DownloadFile('https://raw.githubusercontent.com/3ecurity/3ecurity.github.io/master/test.jpg', 'test1.jpg')}"

2. BitsTransfer

powershell.exe -Command "& {Import-Module BitsTransfer; Start-BitsTransfer 'https://raw.githubusercontent.com/3ecurity/3ecurity.github.io/master/test.jpg' 'test2.jpg'}"

3. Invoke-WebRequest (was added in PowerShell 3.0)

powershell.exe -Command "& {Invoke-WebRequest 'https://raw.githubusercontent.com/3ecurity/3ecurity.github.io/master/test.jpg' -OutFile 'test3.jpg'}"

4. iwr (alias for Invoke-WebRequest since PowerShell 3.0)

powershell.exe -Command “& {(iwr https://raw.githubusercontent.com/3ecurity/3ecurity.github.io/master/test.jpg).Content > test4.jpg}” 由于之前的命令是有问题的，经过google，正确用法如下：

powershell -command "& { iwr https://raw.githubusercontent.com/3ecurity/3ecurity.github.io/master/test.jpg -OutFile test4.jpg }"

5. wget, curl (aliases for Invoke-WebRequest since PowerShell 4.0)

powershell.exe -Command "& {wget 'https://raw.githubusercontent.com/3ecurity/3ecurity.github.io/master/test.jpg' -OutFile 'test5.jpg'}"

powershell.exe -Command "& {curl 'https://raw.githubusercontent.com/3ecurity/3ecurity.github.io/master/test.jpg' -OutFile 'test6.jpg'}"
